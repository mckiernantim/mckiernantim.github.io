Familr

Running Dungeons and Dragons games is hard. Really hard. Familr is a front end GUI built with Angular 6 and hosted on Google Firebase. Consumes the official D&D API via AJAX call and allows the user to track their multiple insances of monsters, spells, or whatever you need to kill your friends for fun.
